GET /api​/v4​/join​/{token}​/preview¶

Invitation information.

{
  "ok": true,
  "result": {
    "icons": {
      "blurhash": "string",
      "color": "string",
      "letters": "string",
      "lg": {
        "height": 0,
        "url": "string",
        "width": 0
      },
      "sm": {
        "height": 0,
        "url": "string",
        "width": 0
      },
      "stub": "string"
    },
    "name": "string",
    "uid": "string"
  }
}

Try it!

Parameters

  • token – Invite token to preview.

Response JSON Object

  • ok (boolean) – True if no error occured.

  • result (object) – Invite information.

GET ​/api​/v4​/ping¶

Ping the server.

Example response:

HTTP/1.1 200 OK
Content-Type: text/javascript

{
  "ok": true,
  "result": "pong"
}

Try it!

Response JSON Object

  • ok (boolean) – True if no error occured.

  • result (string) – Set to "pong".

Status Codes

  • 200 OK – No error
